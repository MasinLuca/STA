Operatore And

L'operatore AND logico produce il valore 1 se entrambi gli operandi hanno valori diversi da zero. Se uno degli operandi è uguale a 0 il risultato è 0. Se il primo operando di un'operazione AND logica è uguale a 0, il secondo operando non viene valutato.



Tavola Di Verità

A B A e B
0 0 0
1 0 0
0 1 0
1 1 1
